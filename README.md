# 🎬 Movie Ticket Booking System

A full-stack web application inspired by BookMyShow that allows users to browse movies, select seats, and book tickets in real time.

---

## 🚀 Features

* 🎥 Browse available movies
* 🕒 View show timings
* 💺 Select seats
* 🧾 Book tickets with confirmation
* 🔐 User authentication (Login/Signup)

---

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Express)
* Database: MySQL
* Tools: Git, GitHub

## ⚙️ System Flow

1. User logs in or signs up
2. Selects a movie
3. Chooses show timing
4. Selects seats
5. Booking is stored in database

---

## 🧠 System Design Concepts

* Backend built using Express.js
* REST APIs used for communication
* MySQL database used to store users and bookings
* Basic seat selection logic implemented
